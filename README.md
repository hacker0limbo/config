    - auto rename tag
    - code spell checker
    - emacs friendly keymap
    - eslint
    - gitLens
    - markdown all in one
    - material icon theme

    - prettier
- sublime

  - `.sublime-settings` 个人配置
  - `.sublime-keymap` 键位配置
  - 插件:
    - emacs pro essentials
    - typescript
    - package control
    - jsprettier
    - dracula
